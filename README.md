# ag
 ag 是一个用go实现的命令行下查看每个网卡详细流量的小工具
![](./screen.png)

# 安装

```golang
go get -v github.com/u35s/ag
```

# 使用
需要root用户执行

```
tab键切换网卡
down向下滚动一条记录
left回到顶部
```
